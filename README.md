# Tunisian Restaurant Website

## Project Overview
This project is a website for a Tunisian restaurant. The website allows users to learn about the restaurant, view the menu, contact the restaurant, and make table reservations online.

## Website Pages
- Home – Welcome message and navigation links  
- About – Information about the restaurant and its Tunisian cuisine  
- Menu – List of food items with prices  
- Contact – Restaurant email, phone number, and message form  
- Reservation – Table reservation form (name, date, number of guests)
- Admin – list of all reservation table with validation status

## Admin Role
The admin represents a restaurant worker.  
- Reviews reservation requests  
- Validates or rejects reservations  
- Manages reservation status (no login system)

## Technologies Used
- HTML  
- CSS   
- Javascript (Firebase connection)
